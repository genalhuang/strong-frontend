#### 键索引计数法
```java
int N = a.length;
String[] aux = new String[N]; 
int[] count = new int[R+1];
// 计算出现频率
for (int i = 0; i < N; i++) 
 count[a[i].key() + 1]++; 
// 将频率转换为索引
for (int r = 0; r < R; r++) 
 count[r+1] += count[r]; 
// 将元素分类
for (int i = 0; i < N; i++) 
 aux[count[a[i].key()]++] = a[i]; 
// 回写
for (int i = 0; i < N; i++) 
 a[i] = aux[i];
```
